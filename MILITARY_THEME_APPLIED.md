### 🔧 Customization Options

If you want to adjust anything:

**Change Colors**: Edit `military_theme.css`
```css
/* Main green color */
#10b981 → Your preferred color

/* Background */
#1a2332 → Your preferred dark color
```

**Change Emblems**: Update image URLs in index.html
```html
<!-- Left emblem (DRDO) -->
<img src="YOUR_LEFT_EMBLEM_URL" class="military-emblem">

<!-- Right emblem (India) -->
<img src="YOUR_RIGHT_EMBLEM_URL" class="india-emblem">
```

**Change Header Text**:
```html
<h1>🌊 MARINE DEFENSE CYBER SHIELD</h1>
<p>Underwater Image Enhancement System</p>
```
